# Voice Active Prompter v1.0.0

첫 정식 안정 릴리즈입니다.  
이번 버전은 데스크톱 실사용 흐름과 실시간 음성 인식 정확도/안정성 개선에 초점을 맞췄습니다.

## What's New

- FastAPI + React + Desktop 통합 실행 흐름 안정화
- 마이크 기반 실시간 전사와 대본 자동 진행 로직 개선
- 발표용 라이브 자막창, 전체화면 모드, 미러(좌우 반전) 모드 제공
- Windows 실행 스크립트(`setup_windows.bat`, `run_*.bat`) 정리
- `.env.example` 기반 초기 설정 절차 표준화
- README 및 스크린샷 문서 보강

## Included

- Backend API: `/api/health`, `/api/transcribe`, `/api/progress`, `/api/settings/defaults`
- Desktop App: `python -m app`
- Frontend Dev App: `npm run dev`

## Quick Start

1. Windows에서 `setup_windows.bat`를 1회 실행합니다.
2. `run_desktop.bat`로 데스크톱 앱을 실행합니다.
3. 마이크를 선택하고 대본을 읽어 자동 진행을 확인합니다.
